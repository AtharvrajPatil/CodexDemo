# CodexDemo

Comet Dodge is a simple browser arcade game. Pilot your ship and dodge falling comets for as long as you can.

## Play

1. Open `index.html` in your browser.
2. Click Start.
3. Move with `A`/`D` or the left/right arrow keys.

## Files

- `index.html` — markup and layout
- `styles.css` — visual theme
- `script.js` — game logic
